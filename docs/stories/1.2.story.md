# Story 1.2 — Scraper Local (CAPES + DOU + Top 20 IES)

**Epic:** EPIC-001 PosGrad Tracker
**Story ID:** 1.2
**Status:** Draft
**Agente:** @pieter-levels
**Estimativa:** 4 dias
**Depende de:** 1.1 (não bloqueia, pode rodar em paralelo)

---

## Descrição

Como desenvolvedor, quero um pipeline de scraping em Python que coleta programas e editais do CAPES Sucupira, DOU federal e Top 20 IES, salvando tudo em arquivos JSON locais validados — sem banco de dados ainda.

## Acceptance Criteria

- [ ] **AC1:** Estrutura de diretórios criada: `scrapers/`, `data/`, `logs/`
- [ ] **AC2:** `scrapers/capes_sucupira.py` — scraper que gera `data/programas.json` com campos: `codigo_capes`, `nome`, `grande_area`, `area_capes`, `nivel`, `ies_nome`, `ies_sigla`, `estado`, `nota_capes`
- [ ] **AC3:** `scrapers/dou_federal.py` — scraper que gera `data/editais_dou.json` com campos: `titulo`, `descricao`, `link_edital`, `data_publicacao`, `fonte`
- [ ] **AC4:** `scrapers/ies_editais.py` — scraper configurado para Top 20 IES, gera `data/editais_ies.json`
- [ ] **AC5:** `scrapers/main.py` — orchestrador que roda os 3 scrapers em sequência com `python scrapers/main.py editais`
- [ ] **AC6:** Rate limiting implementado (≥ 1 segundo entre requests por domínio)
- [ ] **AC7:** Logging em `logs/scraper.log` com timestamp, fonte, registros coletados e erros
- [ ] **AC8:** `requirements.txt` com todas as dependências
- [ ] **AC9:** Resultado mínimo após rodar: 500+ programas em `programas.json`, 10+ editais em `editais_dou.json`
- [ ] **AC10:** Script não quebra se uma fonte falhar — continua as demais e loga o erro
- [ ] **AC11:** `data/` adicionado ao `.gitignore` (não versionar JSONs grandes)
- [ ] **AC12:** `README-scraper.md` documentando como rodar localmente

## Scope

**IN:**
- CAPES Sucupira (programas)
- DOU federal (editais)
- Top 20 IES por volume de programas
- Output em JSON local

**OUT:**
- Salvar no Supabase (Story 1.5)
- GitHub Actions cron (Story 1.5)
- Top 100 IES completo (incrementar na Story 1.5)
