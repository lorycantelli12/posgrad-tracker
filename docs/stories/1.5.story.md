# Story 1.5 — Integração Supabase

**Epic:** EPIC-001 PosGrad Tracker
**Story ID:** 1.5
**Status:** Draft
**Agente:** @posgrad-chief (coordena todos os agentes)
**Estimativa:** 5 dias

## Descrição

Como desenvolvedor, quero conectar todas as peças validadas nas stories anteriores ao Supabase — migrando dados dos JSONs para o banco, integrando auth real, transformando scripts Python em Edge Functions automáticas e ativando o ciclo completo de scraper → matching → notificação.

## Acceptance Criteria

### Banco e Auth
- [ ] **AC1:** Projeto Supabase criado com schema SQL completo (`migrations/001_initial_schema.sql`)
- [ ] **AC2:** Supabase Auth configurado (Google OAuth + email/senha)
- [ ] **AC3:** Dados dos JSONs importados para o banco (`python migrate/import_json.py`)
- [ ] **AC4:** RLS (Row Level Security) configurado — usuário só vê seus próprios dados
- [ ] **AC5:** Frontend substituindo mock data por queries Supabase reais

### Preferências e Matching
- [ ] **AC6:** Tabela `user_preferences` funcionando — onboarding salva preferências reais
- [ ] **AC7:** Edge Function `run-matching` implementada e testada
- [ ] **AC8:** Cold start funcionando — usuário novo vê matches na hora após onboarding
- [ ] **AC9:** Deduplicação de matches funcionando (mesmo edital não aparece 2x)

### Notificações Automáticas
- [ ] **AC10:** Edge Function `send-notifications` integrada com OneSignal e Supabase
- [ ] **AC11:** Limite diário implementado (máx 3 pushes/dia/usuário)
- [ ] **AC12:** Edge Function `send-lifecycle-email` com Resend (welcome + first_match + digest)
- [ ] **AC13:** Tabela `notification_log` registrando todos os envios

### Scraper Automático
- [ ] **AC14:** Scraper adaptado para salvar diretamente no Supabase (além do JSON local)
- [ ] **AC15:** GitHub Actions configurado (cron diário às 6h BRT)
- [ ] **AC16:** Após scraper, Edge Function de matching é invocada automaticamente

### Ciclo Completo
- [ ] **AC17:** Ciclo testado end-to-end: scraper → banco → matching → push → email
- [ ] **AC18:** Conta de usuário real criada, preferências configuradas, push recebido
