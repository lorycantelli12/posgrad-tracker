## Descrição

Como usuário que quer estudar/pesquisar no exterior, quero que o sistema monitore bolsas internacionais reais em universidades da Alemanha, EUA, Austrália e Canadá (além das fontes brasileiras CAPES/CNPq/Fulbright), para receber alertas de oportunidades que realmente existem — sem depender de mock data.

## Contexto

O scraper `scrapers/internacional.py` atual usa o DOU como fonte e cai no mock quando retorna < 5 resultados. Para cobrir as principais universidades do exterior, a estratégia mais confiável não é scraping direto site-a-site (cada universidade tem estrutura diferente, quebra frequentemente), mas usar **portais agregadores por país** que centralizam oportunidades de múltiplas instituições:

| País | Portal Agregador | Cobertura |
|------|-----------------|-----------|
| Alemanha | DAAD (daad.de) + EURAXESS Germany | 400+ universidades alemãs |
| EUA | NSF Graduate Fellowships + ProFellow + USAJobs Research | Federais + top privadas |
| Austrália | Universities Australia + Study Australia | Grupo dos 8 + demais |
| Canadá | EduCanada (educanada.ca) + CGS/NSERC | Federais + provinciais |
| Europa geral | EURAXESS Jobs (euraxess.ec.europa.eu) | 40 países |

Além dos agregadores, as **top 5 universidades por país** têm páginas de bolsas estruturadas o suficiente para scraping direto.

- Tradução automática de editais em inglês

### Estratégia por tipo de fonte

| DAAD | HTML estático | httpx + BS4 |
| EURAXESS | REST API + JSON | httpx |

### Sazonalidade
- NSF GRFP: inscrições em agosto/setembro — fora deste período retorna 0 real → mock
- Australia Awards: inscrições em fevereiro/março
- DAAD: editais abertos o ano todo (bolsas mensais e anuais)
- EduCanada: editais federais em janeiro/fevereiro

### Rate limiting
- 2s entre requests por domínio
- Playwright: headless, sem JavaScript desativado
- User-Agent: `Mozilla/5.0 (compatible; PosGradTracker/1.0; +https://posgrad-tracker.vercel.app)`
