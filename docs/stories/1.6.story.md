# Story 1.6 — Polish e Launch

**Epic:** EPIC-001 PosGrad Tracker
**Story ID:** 1.6
**Status:** Draft
**Agente:** @samuel-hulick + @posgrad-chief
**Estimativa:** 3 dias
**Depende de:** 1.5 (ciclo completo funcionando)

---

## Descrição

Como product owner, quero preparar o app para receber os primeiros usuários reais — com landing page definitiva, tratamento de erros, configurações de notificação e domínio customizado.

## Acceptance Criteria

### Landing Page
- [ ] **AC1:** Landing page definitiva com value prop clara, sem referência a "dados mock"
- [ ] **AC2:** Seção "Como funciona" com 3 passos (Configure → Monitoramos → Você é notificado)
- [ ] **AC3:** CTA principal funcionando (leva ao cadastro real)

### UX e Estados
- [ ] **AC4:** Todos os estados de erro tratados com mensagem amigável (falha de rede, timeout, etc.)
- [ ] **AC5:** Loading states em todas as telas que fazem queries (spinner ou skeleton)
- [ ] **AC6:** Estado vazio consistente em todas as listas
- [ ] **AC7:** Página de preferências editáveis (usuário pode mudar área/estado após onboarding)

### Configurações de Notificação
- [ ] **AC8:** Tela de configurações: pausar notificações (7/30/90 dias)
- [ ] **AC9:** Tela de configurações: alterar frequência do digest (diário/semanal/nunca)
- [ ] **AC10:** Botão de cancelar conta com confirmação

### Performance e PWA
- [ ] **AC11:** Lighthouse score ≥ 80 em Performance, Accessibility e Best Practices
- [ ] **AC12:** `public/manifest.json` configurado (ícone, nome, theme_color)
- [ ] **AC13:** App instalável via browser (PWA) no iOS e Android

### Domínio e Deploy
- [ ] **AC14:** Domínio customizado configurado na Vercel (ex: posgrad.app ou subdomínio)
- [ ] **AC15:** SSL ativo no domínio customizado
- [ ] **AC16:** Vercel Analytics ativo

### Alpha
- [ ] **AC17:** 5 usuários alpha convidados e cadastrados
- [ ] **AC18:** Feedback coletado de pelo menos 3 usuários alpha
- [ ] **AC19:** Bugs críticos reportados pelo alpha corrigidos

## Scope

**IN:**
- Landing page final
- Tratamento de erros
- Configurações de notificação
- PWA
- Domínio customizado
- Alpha com 5 usuários

**OUT:**
- App nativo iOS/Android
- Busca por texto livre
- Recomendação de orientadores
- Dashboard de analytics para o admin
