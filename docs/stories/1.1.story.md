## Descrição

Como desenvolvedor, quero construir toda a UI do posgrad-tracker com dados hardcoded para validar o design e o fluxo de onboarding antes de conectar qualquer backend.

## Contexto

Esta é a primeira story. Não há banco de dados, autenticação real nem scraper ainda. O objetivo é ter a UI funcionando de ponta a ponta com dados falsos para validar:
- O fluxo de onboarding (6 telas)
- O design da tela de editais
- A experiência de "zero matches"
